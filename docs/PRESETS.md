# Presets

## unity-game
Unity game project with C# scripting. Includes game dev agents (gameplay, AI, UI programmers), Unity-specific skills, and C# code standards.

## threejs-game
Browser-based 3D game with Three.js. Includes shader skills, frontend dev tooling, and WebGL performance standards.

## saas-web
Full-stack SaaS web application. Includes fullstack dev, database, UI/UX agents, auth and payment skills, and web security standards.

## api-service
Backend API service / microservice. Includes backend dev, database, security agents, and API design standards.

## custom
Minimal preset — the interview asks additional questions to determine what skills, agents, and standards to install.
